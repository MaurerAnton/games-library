# Games Library

4 pure browser games — no dependencies, no build step, just open in a browser.

- **Synthwave Typing** — neon typing trainer with falling words
- **Constellation Quiz** — learn constellations by connecting stars
- **Emotional Tapestry** — year-long mood tracker with poster export
- **Glitch Art Generator** — apply glitch effects to images

## Run Locally

```bash
git clone https://github.com/MaurerAnton/games-library.git
cd games-library
python3 -m http.server 8000
```

Then open:
- http://localhost:8000/typing/
- http://localhost:8000/constellations/
- http://localhost:8000/tapestry/
- http://localhost:8000/glitch/

## Direct Open

Just double-click any `index.html` file — no server required.
